# Student Performance Prediction

A machine learning project developed using Python and Logistic Regression to predict whether a student may get high or low marks based on study hours.

## Features
- Takes study hours as input
- Predicts student performance
- Shows a graph of study hours vs result
- Calculates model accuracy

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
